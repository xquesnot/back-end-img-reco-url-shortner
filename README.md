# back-end-img-reco-url-shortner
Back End URL SHORTNER

## METHODS
- create /POST
- get /GET
- list /GET
- delete /DELETE
- update /PUT

## HOW DEPLOY ?

use command : serverless deploy

Before deploy, you must install plugins :

- serverless-bundle
- serverless-offline
- serverless-dotenv-plugin 

Use commande : serverless plugin install -n plugin-name
